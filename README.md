# Алгоритмический планировщик физической нагрузки

Веб-приложение для персонализированного планирования тренировок с автоматической кластеризацией упражнений на основе нейросетевых векторных представлений.

## О проекте

Разработан в рамках научной статьи по физической культуре. Система позволяет студенту получить персонализированный тренировочный план под выбранную цель, визуализировать автоматическую кластеризацию 50 упражнений с помощью нейросетевых эмбеддингов и алгоритма KMeans, отслеживать прогресс во времени с сохранением истории тренировок, а также избежать травмоопасных рекомендаций, которые часто выдают генеративные модели типа DeepSeek или ChatGPT.

## Технологии

Язык программирования: Python 3.13+. Веб-фреймворк: Streamlit. Нейросетевые эмбеддинги: Sentence-Transformers (модель paraphrase-multilingual-MiniLM-L12-v2). Кластеризация: Scikit-learn (KMeans, PCA). Обработка данных: Pandas. Визуализация: Matplotlib.

## становка и запуск

Клонируйте репозиторий: git clone https://github.com/ваш-username/fitness-planner-ai.git и перейдите в папку проекта: cd fitness-planner-ai. Установите зависимости: pip install -r requirements.txt. Запустите приложение: streamlit run app.py. После этого приложение откроется в браузере по адресу http://localhost:8501

## 🎮Как пользоваться

Вкладка «Планировщик тренировок»: выберите целевое упражнение из списка, укажите текущий максимум повторений (0 — если не умеете выполнять ни разу), отметьте доступный инвентарь, выберите частоту тренировок и нажмите кнопку «Построить тренировочный план». Если текущий уровень равен 0, система построит цепочку подводящих упражнений на 6 недель. Если текущий уровень больше или равен 1, система построит план прогрессии с увеличением нагрузки на 5-10% каждую неделю.

Вкладка «Кластеризация упражнений»: здесь представлен точечный график, где цвет точки определяет принадлежность упражнения к одному из 7 кластеров, выявленных алгоритмом KMeans. Справа от графика находятся характеристики каждого кластера: преобладающая группа мышц, средняя сложность и примеры упражнений. Ниже расположена полная таблица всех упражнений с указанием номера кластера.

Вкладка «Мой прогресс»: здесь отображается график динамики результатов, таблица истории тренировок и кнопка очистки истории. Данные сохраняются автоматически при нажатии кнопки «Отметить неделю как выполненную» в планировщике.

## База упражнений

Файл exercises.csv содержит 50 упражнений, аннотированных по следующим полям: name — название упражнения, target_muscle — целевая группа мышц, secondary_muscles — вспомогательные мышцы, difficulty — уровень сложности от 1 до 5, equipment — необходимый инвентарь, description — текстовое описание техники выполнения.

## Как работает кластеризация

Для каждого упражнения формируется текстовый дескриптор, объединяющий название, целевую группу мышц, уровень сложности, инвентарь и описание. Модель Sentence-Transformer преобразует этот текст в числовой вектор размерностью 384. Все векторы нормализуются, после чего алгоритм KMeans группирует их в 7 кластеров. Для визуализации применяется метод главных компонент, который проецирует 384-мерные векторы на двухмерную плоскость.

## Перспективы развития

Ключевыми направлениями для дальнейшего развития системы являются расширение базы упражнений до 200 и более позиций, эксперименты с другими алгоритмами кластеризации такими как DBSCAN и иерархическая кластеризация, внедрение адаптивной подстройки нагрузки через методы машинного обучения с подкреплением, а также педагогическое тестирование системы в образовательном процессе кафедры физического воспитания.
